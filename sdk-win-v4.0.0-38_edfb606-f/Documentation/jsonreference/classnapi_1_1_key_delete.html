Detailed Description

Request certain keys to be deleted from the provision.

Request certain keys to be deleted from the provision. If possible the keys are removed from the Nymi Band as well and the local NEA provision.

Deleting the Roaming Authentication setup key simply removes the setup information from the currenty provision. This will not remove the capability of Roaming Authentication with the deleted setup from the Nymi Band. Many NEAs routinely delete the provision that was used to setup Roaming Authentication, in which case explicitly deleting this key is not necessary.j

Currently there is no direct way to disable Roaming Authentication on a Nymi Band, however there are two steps that will accomplish the same effect:

  1. tell the Roaming Authenticator service to delete the raKeyId and associated user data
  2. setup Roaming Authentication again with the same partner public key, and don't inform the Roaming Authenticator service
