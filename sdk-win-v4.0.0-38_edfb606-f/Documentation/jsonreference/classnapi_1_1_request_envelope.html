Detailed Description

The envelope wrapping the JSON API request.

When a request is made to the JSON API it is wrapped in this JSON object. The response from the JSON API is wrapped in the same object with some additional attributes defined.

The exchange attribute is an arbitrary string determined by the NEA that can be used to match this request to it's response.
