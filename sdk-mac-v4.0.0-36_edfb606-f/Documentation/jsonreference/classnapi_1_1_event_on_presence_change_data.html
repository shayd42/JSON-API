Detailed Description

onPresenceChange notification

See the detailed discussion of the 'presence' status.

The notification will contain and associate the pid only if it is known. It is possible that on first appearance, and after provisioning, the pid is unknown for a short time (< 1s) and some notifications may be reported without the pid.

NOTE: Until either this or the onFoundChange notification is received with a pid specified, the NEA must not assume that any provision is present (this is especially true just after provisioning is complete).
