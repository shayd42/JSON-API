Detailed Description

Initiate the Roaming Authentication process.

The only information required to start the Roaming Authentication process is the tid of the Nymi Band that the NEA would like to authenticate. In particular, no provision is required.

Roaming Authentication is a multiple request/response/notifcation process, and the exchange ID is used to associate the messages (essential if the NEA is trying to Roaming Authenticate more than one Nymi Band at a time).
